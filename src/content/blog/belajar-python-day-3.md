Hari ini adalah day 3, dan ini yang saya pelajari di hari ini:

### If

If(jika), jika kondisi terpenuhi maka kode bisa dijalankan.

contoh:

```
ayam = 20

if ayam > 20:
 print("banyak")

print("selesai")
```

kode tersebut menghasilkan output selesai, dan print "banyak" tidak dijalankan. kenapa? karena angka 20 tidak lebih besar dari 20, jika ingin menghasilkan "banyak" maka menggunakan simbol >= (lebih besar atau SAMA dengan)

### else

(kalau tidak), jika kondisi if tidak terpenuhi, maka else akan dieksekusi. contoh:

```
ayam = 20

if ayam > 20:
 print("banyak")
else:
 print("sedikit"

print("selesai")

```





outputnya akan menghasilkan "sedikit" dan "selesai", kenapa?  karena kondisi if tidak terpenuhi, oleh karena itu baris else dieksekusi.
