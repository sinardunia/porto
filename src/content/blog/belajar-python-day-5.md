### list di dalam python

List, sebuah wadah yang disimpan dalam [], mirip array pada bahasa lain. Bedanya, bisa menyimpan berbagai tipe data.

Ada beberapa catatan terkait penggunaan list ini, Yaitu:

- cara pembacaan (indexing) dimulai dari nol
- fungsi append() untuk menambah data pada list akhir.
- len() fungsi bawaan untuk menghitung total isi list ada berapa.
- remove() menghapus berdasarkan nilai, mesti diisi karena bukan dihapus berdasarkan urutan.
- assigment bisa mengubah isi list, contoh:

```
presiden_konoha = ["joko", "prawo", "naruto"]
presiden_konoha[0] = "gibran"
```

Artinya, gibran menimpa joko pada urutan pertama (karena dimulai dari nol maka  ist pertama jadi menggunakan simbol nol)

Hal ini disebut dengan assigment, yaitu mengganti nilai (menimpa) pada index tertentu. hal ini juga berlaku pada variabel, seperti:

```
umur = 1
umur = 2
```
