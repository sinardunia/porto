### list of dictionary

sebuah list, berisi dictionary. disebut list of dictionary.

contoh:

```
produk = [
    {"nama": "Pena", "harga": 5000, "stok": 20},
    {"nama": "Buku", "harga": 15000, "stok": 5},
    {"nama": "Pensil", "harga": 3000, "stok": 0}
]
```

kenapa kita membutuhkan itu?

karena, software modern hampir membutuhkan list of dictionary. sering digunakan di:

- api
- json
- database query

---

- list diibaratkan seperti rak buku 
- dan judul buku beserta halaman dan nama penulis adalah dictionary-nya.
- list of dictionary adalah data rak-rak berisi buku-buku dengan nama penulisnya lengkap
