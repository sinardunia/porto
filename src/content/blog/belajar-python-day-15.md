### HTTP

> apa itu http?

HTTP kepanjangan dari **Hypertext Transfer Protocol.**

**HTTP adalah "bahasa" yang dimengerti untuk bertukar data dari server manapun, di dunia.**

### Kenapa kita membutuhkan HTTP?

Karena HTTP dipakai oleh Browser, backend, mobile apps, sampai ke automation script. Semuanya menggunakan HTTP.
