# Code of Conduct

## Our Standards

We are committed to providing a friendly, safe, and welcoming environment for all:

- **Be respectful** — disagree without being disagreeable
- **Be constructive** — criticism is welcome when paired with suggestions
- **Be inclusive** — welcome newcomers and explain context patiently
- **Stay focused** — keep discussions relevant to the project

## Unacceptable Behavior

- Harassment, discrimination, or intimidation of any kind
- Trolling, insulting/derogatory comments, or personal attacks
- Publishing others' private information without permission
- Dismissing or repeatedly shutting down valid contributions

## Reporting Issues

If you experience or witness unacceptable behavior:

1. **GitHub**: Open a private security advisory if sensitive
2. **Public issues**: For general concerns, open a regular issue and mention a maintainer

All reports will be handled confidentially and addressed within 72 hours.

## Scope

This code applies to:
- GitHub issues, pull requests, and discussions
- Any communication related to the project
- Representation of the project in public spaces

## Enforcement

Maintainers may take any action they deem appropriate, including:
- Warning the offender
- Temporary or permanent ban from the project
- Removal of contributions
